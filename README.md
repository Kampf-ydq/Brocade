# 项目锦集

## 装甲识别
>名称：ROBOMASTER 2018 机甲大师赛  
>时间：2018年05月  
>描述：此次比赛为期5天，ROBO INNVOATION实验室参赛队伍代表沈阳工业大学，前往中国石油大学参加北部赛区。而我的任务则是利用OpenCV开源框架对图像做相应处理，
完成对敌方装甲（蓝色、红色）的识别，发送识别坐标，实现自动瞄准射击。
